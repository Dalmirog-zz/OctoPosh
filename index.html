OctoPosh

A Powershell module providing cmdlet to do basic tasks on Octopus Deploy.

What?

OctoPosh is a Powershell module that provides cmdlet to do basic tasks on Octopus Deploy. All the cmdlets use a combination of calls to the Octopus REST API and some occasional uses of the Octopus.Client(which in the end, also uses the REST API).

The goal of this module is for it to be the go-to place to programatically get information about your Octopus instance, as well as to automate tasks to help you get your Deployment environment up and running fast and easy. If you go to Octopus' super amazing support site, you'll notice that there are plenty of questions of the kind How do I do [this] using the REST API?. One of the objectives of this module is to create tools to help the user be able to do things in Octopus from Powershell in a rather intuitive way.

How?

  1. Get Chocolatey.
  2. Run:
    cinst Octoposh -source "http://www.nuget.org/api/v2/"
